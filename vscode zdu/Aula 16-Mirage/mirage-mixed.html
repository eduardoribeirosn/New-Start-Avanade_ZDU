Front-end

Front-end é a parte da construção de websites e aplicações que o usuário final vê e interage diretamente. Envolve a implementação de design, interatividade e a experiência do usuário através de tecnologias como HTML, CSS e JavaScript.

Banco de dados

Um banco de dados é um sistema de armazenamento de dados que permite a inserção, consulta, atualização e administração de dados. Ele é essencial para gerenciar grandes quantidades de informações de forma organizada, eficiente e segura.
